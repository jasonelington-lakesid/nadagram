# nadagram

A Flutter Project that imitates Instagram.

## TechStack:
- Flutter: 3.44.1

## Depedencies:
- hive_ce: 2.19.3
- hive_ce_flutter: 2.3.4
- Google_Fonts: 6.3.0
- build_runner: 2.4.15
- hive_ce_generator: 1.9.0
- http: 1.5.0

## Limitations:
- Only one user
- Title length is limited
- No one can comment
- Can not share the post
- No dislikes
- Can not edit post
- Image requires internet connection
- Handler for POST button is not built properly
